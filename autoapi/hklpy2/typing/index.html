hklpy2.typing#

Package-level type aliases and typed dictionary structures for hklpy2.

Consolidates the type aliases previously defined in hklpy2.misc (see issue #252) alongside the TypedDict subclasses introduced in issue #233. hklpy2.misc re-exports everything defined here for backward compatibility.

Simple type aliases

AnyAxesType

Any of these types are used to describe both pseudo and real axes.

AxesArray

Numpy array of axes values.

AxesDict

Dictionary of axes names and values.

AxesList

List of axes values.

AxesTuple

Tuple of axes values.

BlueskyPlanType

Type of a bluesky plan.

INPUT_VECTOR

Acceptable forms of vector input for zones, ...

KeyValueMap

Dictionary for configuration and other.

Matrix3x3

mutable orientation & rotation matrices.

NamedFloatDict

dictionary of named floats.

NUMERIC

Either integer or real number.

TypedDict structures

ConfigHeaderDict

Typed structure for the configuration _header block.

Solver-internal typed structures (ReflectionDict, SampleDict, SolverMetadataDict) live in hklpy2.backends.typing. Solver-specific metadata extensions (e.g. HklSolverMetadataDict) live in the solver’s own module (e.g. hklpy2.backends.hkl_soleil).
